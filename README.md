# горячие клавиши
### выделить все:
Cntrl+A
### копировать:
Cntrl+C
### вставить:
Cntrl+V
### сохранить:
Cntrl+S
### вырезать:
Cntrl+X
### отменить действие:
Cntrl+Z
# работа с git
### Скачать проект
```bash
git clone <URL>
```
### учесть все файлы 
```bash
git add .
```
### сохранить commit
```bash
git commit -m "...."
```
### отправка в облако
```bash
git push
```
